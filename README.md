# Portfolio de Angel Cardenas

Portafolio profesional bilingue construido con React y Vite.

## Desarrollo

```bash
npm install
npm run dev
```

## Validacion

```bash
npm run lint
npm run build
```

## Despliegue

El sitio se publica mediante GitHub Actions en:

`https://watenshi.github.io/portfolio-angel-cardenas/`
